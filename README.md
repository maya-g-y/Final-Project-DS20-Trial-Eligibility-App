# Trial Screening Agent (MIMIC demo)

Streamlit demo to assist research coordinators with patient prescreening:
- Add a trial (inclusion/exclusion criteria)
- Upload/select patients
- Agent uses RAG + tools to decide eligibility with evidence
- Optional LLM-as-a-judge evaluation vs gold standard

## Run (local)
1. Create venv and install requirements
2. `streamlit run app.py`
